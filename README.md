# Restaurant System Client

Frontend for the Restaurant System project. The application is built with React, Vite, Material UI, and Axios.

## Tech Stack

- React
- Vite
- JavaScript
- Material UI
- Axios

## Features

- Shared layout and navigation
- Material UI theme setup
- Axios API client
- Category API service
- Menu item API service
